总体目标：

通过技术研究与创新，提出和建立八角高接换冠快速改良,高效树冠高楼树冠调控与构建、果期保果壮果等八角提质增效栽培关键技术， 为八角低产林改造、实现现八角栽培提质增效提供技术支撑与应用示范。

主要研究内容

(1)八角高接换冠快速改良技术研究

利用现有八角林， 开展不同树形、不同树龄、不同嫁接时间的八角高接换冠技术研究， 综合考虑树冠恢 复速度、型实品质、采搞成本等经济因素， 提出实现快速改良、丰产优质、节本增效目标的A角商战换 冠技Ac.
(2)八角高接换冠快速改良技术研究

利用现有八角林， 开展不同树形、不同树龄、不同嫁接时间的八角高接换冠技术研究， 综合考虑树冠恢 复速度、型实品质、采搞成本等经济因素， 提出实现快速改良、丰产优质、节本增效目标的A角商战换 冠技Ac.
(3)八角高接换冠快速改良技术研究

利用现有八角林， 开展不同树形、不同树龄、不同嫁接时间的八角高接换冠技术研究， 综合考虑树冠恢 复速度、型实品质、采搞成本等经济因素， 提出实现快速改良、丰产优质、节本增效目标的A角商战换 冠技Ac.
